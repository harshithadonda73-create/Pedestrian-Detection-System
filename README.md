# Vision-Based Pedestrian Detection and Collision Warning System

#  Project Overview
This project is developed using Python, OpenCV, and YOLOv8 to detect pedestrians in real time from video input. The system identifies people
, draws bounding boxes around them, and gives warning alerts when a pedestrian comes too close. It also generates a beep sound to help avoid 
accidents and improve road safety.
# Features
- Real-time pedestrian detection
- Collision warning alerts
- Beep sound notification
- Bounding box display
- Confidence score detection
- AI-based object detection using YOLOv8
# Technologies Used
- Python
- OpenCV
- YOLOv8
- Ultralytics
- Computer Vision
